# 使用FastDeploy部署PaddleSeg模型

## FastDeploy介绍

FastDeploy是一款全场景、易用灵活、极致高效的AI推理部署工具，使用FastDeploy可以简单高效的在10+款硬件上对PaddleSeg模型进行快速部署
