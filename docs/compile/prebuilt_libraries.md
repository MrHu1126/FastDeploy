# FastDeploy 预编译部署库

FastDeploy提供了在Windows/Linux/Mac上的预先编译C++部署库，开发者可以直接下载使用，也可以自行编译代码。

## 环境依赖

- cmake >= 3.12
- g++/gcc >= 8.2
- cuda >= 11.2
- cudnn >= 8.0
