# FastDeploy安装

目前已提供Linux、Windows上的Python Wheel安装包，开发者根据自身需求下载后使用如下命令安装即可，如

```
python -m pip install fastdeploy_python-0.2.0-cp36-cp36m-manylinux1_x86_64.whl
```
